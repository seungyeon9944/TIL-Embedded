## 리눅스와 우분투

### 리눅스
**운영체제**란 ? 하드웨어를 관리하는 프로그램. 사용자와 하드웨어 사이의 중개자
- 서버 
- 임베디드 OS 시장 지배
- 안드로이드는 리눅스로 만들어진 Mobile OS

**임베디드**란 ? **특정 목적을 위해 만들어진 컴퓨팅 시스템** (CPU + 메모리 + 저장장치 + 페리페럴)으로 리눅스 👍

Linux는 OS가 아닌 **OS의 핵심 소스코드 역할인 "커널"** 이다. (OS = App + Shell + 커널)


### 리눅스 배포판
Linux에 App, Shell을 추가하여 만든 하나의 운영체제

### 우분투 (Ubuntu)
리눅스 배포판 중 가장 널리 쓰이는 배포판

## 리눅스 파일시스템 및 Shell Command

### Shell
시스템 사용자와 커널과의 **인터페이스를 제공하는 프로그램**
- 컴퓨터를 켜면 OS가 부팅된 후, 실행되는 **프로그램**
    - 시스템 사용자는 Shell을 통해 커널에 명령을 던짐
    - 시스템 사용자는 Shell을 통해 명령에 대한 결과 확인

1. **CLI Shell** : 글자 기반 인터페이스 (한번 익히면 편하고, 편리한 패키지 관리 도구) - 우분투나 라즈베리파이에서는 Bash
2. GUI Shell : 그래픽 기반 인터페이스 - 우분투에서는 **GNOME 3**, 라즈베리파이에서는 LXDE

Shell은 OS에 반드시 포함되어야 하는건 아님. 그리고 사용자는 Shell이 필요없지만 개발자에게는 Shell이 필요

### Shell Command

**Ctrl + Alt + T (단축 키)** 터미널 창 열기

**Ctrl + Shift + w (단축 키)** 혹은 `$exit`
터미널 창 닫기

`ls`
현재 있는 디렉토리 내 목록 출력

`ls -al`
- a 옵션 : all, 숨김 파일까지 출력
- l 옵션 : list, 리스트 형태로 상세 보기

`cd`
- `cd ..` : 상위 디렉토리로 이동
- `cd "디렉토리 이름"`
- `cd ~` : 사용자 홈 디렉토리로 이동
- `cd -` : 이전 디렉토리로 되돌아가기

`pwd`
현재 디렉토리 확인 

`clear`
화면 지우기

**Ctrl + Insert / Shift + Insert**
복사 / 붙여넣기

**Ctrl + S**
화면 출력 정지 (터미널이 멈춤)

**Ctrl + Q**
터미널 시작

### 리눅스 파일시스템
리눅스 파일시스템은 " / "에서 파일관리가 시작됨. 이를 **Root**라고 함

### 파일 관리 명령어
리눅스에서 **파일 이름은 해당 파일이 위치한 경로까지 포함**
- 절대 경로인 `/home/ssafy/snap`으로 접근하는 게 일반적
- 상태 경로도 가능
- 사용자 홈 디렉토리에 대한 별칭도 가능 `~/snap`

`touch [파일명]`
- 새로운 빈 파일 생성
- 이미 있는 파일이면, 변경된 시간을 현재 시간으로
- `touch ./bts`

`rm [파일명]`
- 파일 삭제
- 리눅스에서 삭제는 돌이킬 수 없음 ! 스냅샷을 사용하자 ..
- `rm ./bts`

`mkdir [디렉토리명]`
- 디렉토리 생성
- p 옵션 : 디렉토리 하위메뉴까지 모두 한꺼번에 생성

`rmdir [디렉토리명]`
- 디렉토리 삭제 (디렉토리 내 파일 있으면 삭제 안됨)
- `rm -r [디렉토리명]`하면 디렉토리 내부 파일까지 모두 삭제 가능

`mv [파일] [옮길 곳]`
이동

`mv [파일] [파일이름]`
이름변경

`cp [파일명] [경로]`
- 파일을 특정 경로로 복사
- r 옵션 : 디렉토리 복사
